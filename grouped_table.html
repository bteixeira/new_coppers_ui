  • This is the Grouped Table view
  • This view allows viewing the total amounts for a group of Spendings that share a facet
  • This view is similar to the Detailed Table view, except that the rows (and maybe columns) will be totals instead of the whole details for every Spending
  • Each row will correspond to a time period; A row can show the total for:
    • A Day
    • A Calendar Week
    • A Month
    • A Year
  • Likewise, the Columns will have one of three modes:
    1) No Grouping
    The following columns are shown:
    • Total for this time period
    • Total number of Spendings for this time period
    • Accumulated total for this time period
    There is a row with the total
    2) Grouping by Account
    • Each column shows the name of an Account
    • There is no accumulated column
    • There is a Total row, with the total for each Account
    3) Grouping by Tag
    • Each column shows the name of a Tag
    • There is no accumulated column
    • There is a Total row, with the total for each Tag
    • Unlike Accounts, a Spending may have multiple tags; therefore, the sum of all cells of a specific row may be larger than the actual total for the time period of that row
  • The same filtering is available as on the Detailed View
